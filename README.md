# 테커 Good Night Hackathon 2nd - 영화 리뷰 앱

디자인은 전혀 신경쓰지 않고, 기능만 구현하는 것을 목표로 합니다.

## Supported Features

- [x] 영화 목록 조회
- [x] 영화 상세 조회
- [x] 영화 수정
- [x] 영화 삭제
- [x] 영화 리뷰 조회
- [x] 영화 리뷰 작성

## How to run

[Backend repository](https://github.com/0BVer/movieReview)를 실행합니다.

이후 다음 명령어를 실행합니다.

```bash
$ npm install
$ npm run dev
```
